# Send Text Message

이 엔드포인트는 **문자 메시지(SMS/LMS/MMS)** 를 발송합니다.

## ⚠️ 주의 사항

- 문자 발송은 **Notifly 문자 자가 발송 설정**이 선행되어야 합니다.  
- 발신 번호는 자가 발송 설정에 등록된 번호로 사용됩니다.  
- API로 발송된 통계는 UI에 즉시 제공되지 않을 수 있습니다. 필요 시 contact@notifly.tech 로 문의하세요.

---

## Endpoint

`POST https://api.notifly.tech/messages/text-message`

---

## Specifications

### Request Body

| Parameter     | Type   | Required | Description                                  |
|---------------|--------|----------|----------------------------------------------|
| projectId     | String | Yes      | Notifly 설정 페이지에서 확인 가능             |
| type          | String | Yes      | 메시지 타입: `sms` \| `mms`                   |
| title         | String | No       | LMS/MMS 제목(해당 타입에서만 사용)            |
| body          | String | Yes      | 메시지 본문                                   |
| recipientList | List   | Yes      | 수신자 목록(최대 1,000명)                      |
| └ recipientNo | String | Yes      | 수신자 전화번호                               |

### Headers

| Parameter    | Value            | Description                         |
|--------------|------------------|-------------------------------------|
| Content-Type | application/json | JSON 요청 본문                      |
| Authorization| some-token       | `POST /authenticate` 로 발급받은 토큰 |

---

## Response Example

```json
{ "success": true }
```

---

## Example Code (Node.js)

```js
// 1) 인증 토큰 발급
const authResponse = await fetch("https://api.notifly.tech/authenticate", {
  method: "POST",
  body: JSON.stringify({ accessKey: "your-access-key", secretKey: "your-secret-key" }),
});
const { data: authToken } = await authResponse.json();

// 2) 엔드포인트/헤더
const url = "https://api.notifly.tech/messages/text-message";
const headers = {
  "Content-Type": "application/json",
  Authorization: authToken,
};

// 3) 요청 본문
const body = {
  projectId: "your-project-id",
  type: "sms",
  body: "Message Body",
  recipientList: [{ recipientNo: "01012345678" }, { recipientNo: "01045671234" }]
};

// 4) 호출
const res = await fetch(url, { method: "POST", headers, body: JSON.stringify(body) });
const result = await res.json();
console.log(result);
```

## Error Examples

| Status Code | Description                  | Example Response                     |
|-------------|------------------------------|--------------------------------------|
| 200         | 요청 성공                    | `{"success":true}`                   |
| 400         | 잘못된 요청(필드/형식 오류) | `{"error":"Bad request: ... "}`      |
| 405         | 지원하지 않는 메서드         | `{"error":"Method not allowed"}`     |
| 413         | 요청 크기 초과               | `{"error":"Payload Too Large"}`      |
| 401         | 인증 실패(토큰 만료/무효)    | `{"error":"Unauthorized"}`           |
| 500         | 서버 내부 오류               | `{"error":"Internal Server Error"}`  |
