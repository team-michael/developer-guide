---

이전에 요청한 캠페인 이벤트 데이터 **내보내기 결과를 조회**합니다. 내보내기 상태와 완료된 경우 다운로드 링크를 제공합니다.

> [TODO: confirm endpoints & fields] 엔드포인트/필드 명세는 예시입니다.

## Path Parameters

| Parameter | Type   | Required | Description                              |
|-----------|--------|----------|------------------------------------------|
| requestId | String | Yes      | 내보내기 요청 ID (생성 시 반환된 값)     |

## Response Examples

### Processing (200 OK)

```json
{
  "requestId": "exp_12345",
  "status": "processing",
  "progress": 45,
  "estimatedCompletionTime": "2025-10-22T15:30:00Z",
  "createdAt": "2025-10-22T15:00:00Z"
}
```

### Completed (200 OK)

```json
{
  "requestId": "exp_12345",
  "status": "completed",
  "progress": 100,
  "fileUrl": "https://exports.notifly.tech/exp_12345.csv",
  "fileSize": 2048576,
  "rowCount": 5421,
  "format": "csv",
  "expiresAt": "2025-10-29T15:30:00Z",
  "createdAt": "2025-10-22T15:00:00Z",
  "completedAt": "2025-10-22T15:28:45Z"
}
```

### Failed (200 OK)

```json
{
  "requestId": "exp_12345",
  "status": "failed",
  "progress": 0,
  "error": "Data processing error: invalid date range",
  "createdAt": "2025-10-22T15:00:00Z",
  "failedAt": "2025-10-22T15:05:00Z"
}
```

## Response Fields

| Field                   | Type   | Description                           |
|-------------------------|--------|---------------------------------------|
| requestId               | String | 내보내기 요청 ID                      |
| status                  | String | `queued`, `processing`, `completed`, `failed` |
| progress                | Number | 진행률 (0-100)                        |
| fileUrl                 | String | 다운로드 링크 (완료 시에만 제공)      |
| fileSize                | Number | 파일 크기 (bytes)                     |
| rowCount                | Number | 내보낸 레코드 수                      |
| format                  | String | 파일 형식 (`csv` 또는 `json`)         |
| error                   | String | 실패 사유 (실패 시에만 제공)          |
| estimatedCompletionTime | String | 예상 완료 시간 (처리 중일 때)         |
| expiresAt               | String | 파일 만료 시간 (완료 시)              |
| createdAt               | String | 요청 생성 시간                        |
| completedAt             | String | 완료 시간 (완료 시에만)               |
| failedAt                | String | 실패 시간 (실패 시에만)               |

## Status Values

| Status     | Description                              |
|------------|------------------------------------------|
| queued     | 대기 중 (처리 시작 전)                   |
| processing | 처리 중                                  |
| completed  | 완료됨 (다운로드 가능)                   |
| failed     | 실패함                                   |

## Export Data Format

### CSV 헤더 예시
```csv
timestamp,campaignId,userId,channel,eventName,meta
```

### JSON 레코드 예시
```json
{
  "timestamp": "2025-10-09T12:34:56Z",
  "campaignId": "cmp_abc",
  "userId": "alice",
  "channel": "kakao_alimtalk",
  "eventName": "message_clicked",
  "meta": { "buttonId": "signup", "utm_source": "kakao" }
}
```

## Error Responses

| Status Code | Description      | Example Response                               |
|-------------|------------------|------------------------------------------------|
| 404         | 요청 ID 없음     | `{"error": "Export request not found"}`       |
| 401         | 인증 실패        | `{"error": "Unauthorized"}`                   |
| 500         | 서버 오류        | `{"error": "Internal Server Error"}`          |

## Notes

- 내보내기 파일은 완료 후 7일간 다운로드 가능합니다
- 대용량 파일의 경우 처리 시간이 오래 걸릴 수 있습니다
- 파일 다운로드는 인증이 필요하지 않은 임시 URL입니다
- 진행률은 대략적인 수치이며 실제와 다를 수 있습니다
