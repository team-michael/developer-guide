080 수신거부 서비스에 등록된 **수신자 목록을 조회**합니다.

> [TODO: confirm endpoints & exact payloads]

## Query Parameters

| Parameter | Type   | Required | Description                                   |
|-----------|--------|----------|-----------------------------------------------|
| projectId | String | Yes      | 프로젝트 ID                                   |
| page      | Number | No       | 페이지 번호 (기본값: 1)                       |
| pageSize  | Number | No       | 페이지당 항목 수 (기본값: 100, 최대: 1000)    |

## Response Example

```json
{
  "items": [
    {
      "recipientNo": "01012345678",
      "blockedAt": "2025-10-10T05:12:00Z"
    },
    {
      "recipientNo": "01056781234", 
      "blockedAt": "2025-10-09T14:30:00Z"
    }
  ],
  "total": 2,
  "page": 1,
  "pageSize": 100
}
```

## Response Fields

| Field       | Type   | Description                    |
|-------------|--------|--------------------------------|
| items       | Array  | 차단된 수신자 목록             |
| recipientNo | String | 차단된 전화번호                |
| blockedAt   | String | 차단 등록 시간 (ISO 8601)     |
| total       | Number | 전체 차단 번호 개수            |
| page        | Number | 현재 페이지 번호               |
| pageSize    | Number | 페이지당 항목 수               |

## Error Responses

| Status Code | Description      | Example Response                               |
|-------------|------------------|------------------------------------------------|
| 400         | 잘못된 요청      | `{"error": "Invalid projectId format"}`       |
| 401         | 인증 실패        | `{"error": "Unauthorized"}`                   |
| 500         | 서버 오류        | `{"error": "Internal Server Error"}`          |
